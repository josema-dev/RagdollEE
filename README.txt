==============RAGDOLL_EDITOR_BASED_ON_BUILD_IN_ESENTHEL_RAGDOLL_CLASS==================
Version 0.0.1 alpha. For test purpose only!
All modes:
	1)ESC - Exit editor
	2)Left CTRL - Hold to rotate camera with mouse. Hold right mouse button to move camera.
	3) Mouse scroll - Zoom camera in and out. (Works also with LCTRL)
Simulation mode:
	1)Right mouse button hold on ragdoll body: drag body. (Causes errors!)
View mode:
	1)Left Shift + Left mouse button click: Select ragdoll actor and display parameters in window.
	(IMPORTAND! Change actor or click on viewport to update parameters and deselect active actor!)
	Pink actor: Parent to selected actor to which joint is created.
	Yellow actor: Selected actor.
	Black dot: Joint start pos.
	Green line: Joint.
	
Disable Ragdoll Draw: Do not draw ragdoll and all related things.
Start Simulation: Start physics update.
Disable mesh draw: Do not draw mesh.
Save params: Save ragdoll params to ragdoll_params.txt
Load params: Load ragdoll params from ragdoll_params.txt and set it into current ragdoll.
IMPORTAND! Deselect all actors to enable button.
